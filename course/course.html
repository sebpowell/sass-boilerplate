HTML & CSS

For Beginners

1 day course, part 1 of 3

Summary

This is part 1 of a 3 part series to teach you how to build web interfaces. This first part teaches the basics of front-end coding - it’s aimed at people with no or very little technical knowledge and prior web experience.

You’ll learn how to build basic web pages. Then, if you’re interested in learning how to build more complex interfaces, parts 2 and 3 may be of interest. I personal’y don’t like frameworks like Bootstrap and Foundation, because, whilst they are easy to use, they do little to advance core knowledge and understanding. If you learn the contents of this course, you’ll be able to pick up bootstrap fairly easily.

Prerequisites

In terms of knowledge none really - as long as you know how to use a computer, you’ll be fine. To save time on the day, please install the following software:

  • Sublime Text 3 (OSX | Windows)
  • Git Client or Command Line
  • XCode

Course Contents

I. Setting up your development envioronment - a short intro to the command line, Sublime Text 3 and Git version-control.

II. Intro to HTML & CSS - what they are, how they work together and what you need to know. Chrome Inspector.

III. Device Types & Models, and Browsers

IV. Building a basic website in HTML & CSS. We’ll break this down into various small components, and cover the following:

  • The most commonly used HTML tags and CSS properties
  • Building a simple navigation system, footer and grid system.
  • Positioning and layout

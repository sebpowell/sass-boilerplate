Display

Forces display properties on an element.

  • inline-block: sets display to display: inline-block
  • block: sets display property to display: block

Line Height

Forces line-height. Can be used on parent or element itself.

  • line-height-double: sets line-height property to line-height: 2
  • line-height-reset: sets line-heu

Text Align

Forces text to align. These classes can either be set on the parent element, or the element itself (e.g. h2 class="text-left")

  • text-left: aligns text to the left
  • text-center: aligns text to the center
  • text-right: aligns text to the right

Text Transform

Forces text to transform

  • text-uppercase: sets text to uppercase, and adds 0.05em letter-spacing.
  • text-capitalize: sets text mode to text-transform: capitalize

Margin

Margin helpers are paritcularly useful when you want to fine tune spacing in between elements, without needing to write custom selectors. I don't use them all the time, but there are many cases where, especifailyl in large. The main disadvatnage of the helper approach is that you lose some control, and can risk pullutiny your markup with too many classes. But in some cases, these do the job, and I find they're apritcularly useful for rapid prototpying. The helper classes are available for both padding and margin, as follows:

  • padding-{direction}, margin-{direction} followed by the unit value
  • .no-margin and .no-padding strips off any padding or margins.

Typography

From a designer's perspective, typography is one of the most critial asepcts of any prokect. Managing typgoraphy well, especially with varying viewports makes it even harder. Typgraohyp in X is managed using a typgoraphy scale map, found in the config file. This method has several advantages, namely that it ensures text-sizes are kept consistent throughout your project. It works in the following way:

  • X includes 7 different standard sizes - 3 sizes above the base font size, and 3 sizes above.
  • The size and line-height is then specified for each type of device, found int he breakpoitns map.
  • To use a size on a new element, use the mixin @responsive-text-size(size).
  • You can speficiy additional breakpoints and sizes in the map.

  • text-x-small
  • text-small
  • text-base
  • text-medium
  • text-large
  • text-x-large

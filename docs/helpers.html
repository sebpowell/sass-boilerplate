Helpers

All helper classes have an !important flat set on them to ensure they overide the element's styling.

Background Color

A background-color helper is generated for the main colours defined in our theme file.

By default, these are:

  • .background-light
  • .background-dark
  • .background-white
  • .background-brand-primary
  • .background-brand-secondary

Display

Adds display property to an element.

  • inline-block: sets display to display: inline-block
  • block: sets display property to display: block

Line Height

Forces line-height. Can be used on parent or element itself.

  • line-height-double: sets line-height property to line-height: 2
  • line-height-reset: sets line-heu

Text Align

Forces text to align. These classes can either be set on the parent element, or the element itself (e.g. h2 class="text-left")

  • text-left: aligns text to the left
  • text-center: aligns text to the center
  • text-right: aligns text to the right

Text Transform

Forces text to transform

  • text-uppercase: sets text-fransform to uppercase (adds 0.05em letter-spacing).
  • text-capitalize: sets text mode to capitalize

Margin

Margin helpers are paritcularly useful to help fine tune spacing in between elements, without needing to write custom selectors. The helper classes are available for both padding and margin, as follows:

  • padding-{direction}, margin-{direction} followed by the unit value
  • .no-margin and .no-padding removes any padding or margins.

Lists

Use .no-bullet to remove bullets, margins and padding.

Typography

For pages that include loads of text (such as articles, blogs etc), additional styels are needed to make the text legible.

Add the class 'typography' to your parent element to add proper spacing to headers, parapggs and list items.

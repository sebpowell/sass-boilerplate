Background Images

  • size (default: cover)
  • position (default: center)
  • repeat (default: no-repeat)
+background-image(cover, center, no-repeat)
.element {
   background-size: cover;
   background-position: center;
   background-repeat: no-repeat;
}

Border Radius

+border-radius(3px)
.element {
   border-radius: 3px;
}

Borders

  • Direction (required): Top | Left | Right | Bottom | All
  • Type (defaults to solid)
  • Size (required)
  • Color: defaults to $color-keyline
+border(left, solid, 1px)
.element {
   border-left: 1px solid #e5f5f5;
}

Sliding Panel

Creates the necessary CSS to create a slid-out panel (mobile nav menu etc). 

+sliding-panel($direction, $width)
.element {
   position: fixed;
   width: 400px;
   left: -400px;
}
.element.is-visible {
left: 0;
}

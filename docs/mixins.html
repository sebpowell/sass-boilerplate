Background Images

  • size (default: cover)
  • position (default: center)
  • repeat (default: no-repeat)
+background-image(cover, center, no-repeat)
.element {
   background-size: cover;
   background-position: center;
   background-repeat: no-repeat;
}

Border Radius

+border-radius(3px)
.element {
   border-radius: 3px;
}

Borders

  • Direction (required): Top | Left | Right | Bottom | All
  • Type (defaults to solid)
  • Size (required)
  • Color: defaults to $color-keyline
+border(left, solid, 1px)
.element {
   border-left: 1px solid #e5f5f5;
}

Box Shadow

+box-shadow(2px 2px 3px 0 rgba(00,00,00,0.4))

Ellipsis

Adds ellpsis to prevent text-wrapping.

Width: defaults to 100%. 

+ellipsis($width)

Margin & Padding

Using multiples of this unit allows us to estlablish a consisten rtthm and consistency throughout the platform. There are cases where you don't necessarily want to use exact multuples (for instance, buttons), but in many cases this does help.

These values can be accessed throughout your code using the sizes() function. 

.primary-header-site {
		padding: sizes(base);
  margin-bottom: sizes(double);
}

Lists

+list(bordered, 2rem, clickable, removable)

Sliding Panel

Creates the necessary CSS to create a slid-out panel (mobile nav menu etc). 

+sliding-panel($direction, $width)
.element {
   position: fixed;
   width: 400px;
   left: -400px;
}
.element.is-visible {
left: 0;
}

Transition

+transition($direction, $width)

Triangle

Creates a triangle, used on tooltips, dropdown menus etc. 

+transition($direction, $width, $color)
